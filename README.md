# LangChain Data Processing Pipeline Practice

## Overview

This project demonstrates the core data processing pipeline used in Retrieval-Augmented Generation (RAG) applications with LangChain.

The pipeline includes document loading, text preprocessing, chunking, embedding generation, and vector database storage using FAISS.

## Features

- Document Loading
- Text Splitting and Chunking
- Embedding Generation
- FAISS Vector Database Creation
- Similarity Search
- RAG Workflow Foundation

## Technologies Used

- Python
- LangChain
- HuggingFace Embeddings
- FAISS
- Sentence Transformers
  
## Learning Objectives

- Understand the RAG data pipeline
- Learn document preprocessing
- Create vector embeddings
- Store and retrieve data using FAISS
- Build the foundation for AI-powered applications
